# 言令 1.0.0

言令 1.0.0 把首版声明式解析器推进为具有稳定规格、结构化错误、资源预算、完整帮助和上下文补全的纯言序 CLI 基础库，同时保留 0.1 的主要声明与解析入口。

## 主要内容

- 严格验证命令、选项、短名、值名、参数和说明，失败声明不留下半项；
- 私有内部规格与深复制快照，默认容器在来源、规格和每次解析间隔离；
- 显式队列验证最多 32 层、1024 节点命令树与最多 64 层默认容器，避免宿主栈溢出；
- 4096 argv、单项 65536 字和总量 1048576 字硬边界；
- 长短项、组合开关、附值、否定、真实`no-`优先、多值、位置、余项和嵌套路径；
- 五项严格解析配置，解析与补全共享帮助、穿插、否定和重复开关语义；
- 模式 v1 普通结果与固定六键尝试结果，错误详情提供稳定`YANLING_`代码；
- 缺字、替换和相邻换位的未知长项建议，无编辑矩阵和无界候选；
- 帮助展示选项类型、必需性、重复性、否定形式、默认摘要、位置参数和完整命令路径；
- shell 无关补全状态机识别子命令、待选项值、位置、帮助和首个无效项。

## 安全与资源

- 零第三方依赖；
- 不申请文件、网络、监听、环境、进程或原生扩展权限；
- 命令规格、默认容器、argv、帮助和候选全部有公开预算；
- 补全不执行 shell、不读取文件系统、不调用用户提供器；
- 帮助只预览长文本默认，非空容器只展示类型和项数；
- 直接解析传播错误，尝试解析保留原错误并隔离帮助生成失败。

## 验证资产

- 14 份规格覆盖正常解析、子命令、配置、补全、帮助、结构化错误、默认隔离和真实最大资源边界；
- 2 个可运行示例覆盖基础解析与子命令补全；
- 1 个确定性负载执行 128 次完整解析和 128 次跨层补全；
- 1 个独立消费者通过路径依赖、`^1.0`范围和`包:言令`导出验证锁图；
- 言序 1.1.6 与当前稳定版分别通过格式、静态检查、树/VM 对照和 Release YXB 构建；
- 最低版本矩阵覆盖 Linux、macOS、Windows；生成 API 和锁图加入漂移门禁。

## 兼容提示

- 解析和尝试结果新增固定字段，完整典快照需要更新；
- 无效或歧义规格现在在声明时拒绝；
- 单值选项重复稳定报错，开关可选择严格重复策略；
- 帮助输出更完整，不承诺与 0.1 逐字相同；
- 稳定错误代码替代消息匹配；
- 补全输入最后一项具有“当前前缀”语义。
